# Gemini Reverse Style Template

Use this prompt in Gemini with the user's reference image.

```text
请作为一名顶级 AI 绘画提示词专家，为我分析这张图片的视觉风格。

任务目标：
提取并反推这张图片的艺术风格，生成一份通用的中文生图 Prompt。这份 Prompt 必须剥离原图中的具体角色、IP、文字、商标、地点、人物身份、服装细节和特定情节，只保留可迁移的美学风格。

请覆盖以下 15 个维度，但不要输出分析过程：
1. 画面风格
2. 画面成分组成
3. 构图方式
4. 分镜类型
5. 光影特质
6. 色调与色彩科学
7. 媒介与材质纹理
8. 情绪与氛围
9. 渲染、拍摄或印刷参数
10. 时代感与文化语境
11. 空间逻辑与透视关系
12. 信息密度与留白
13. 动态状态与瞬时感
14. 后期处理与数字痕迹
15. 符号化特征

输出要求：
1. 只输出一段完整、高水准、可直接用于生图的中文 Prompt。
2. 在 Prompt 的开头或核心位置使用占位符：[在此处替换为您想要生成的主体内容]
3. 用户只替换该占位符，就应能在保持原图质感的同时生成新的主体。
4. 不要输出标题、解释、列表、分析过程或多方案。
5. 不要保留原图中的具体角色、IP、文字、商标、剧情或唯一性构图标识。
```
