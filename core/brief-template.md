# Research Brief Template

Use this template when the operator gives a vague request.

## Minimum Brief

- Product direction:
- Search keywords or synonyms:
- Price band:
- MOQ preference:
- Preferred supplier region:
- Must-have traits:
- Must-avoid traits:
- Notes about quality, factory type, packaging, or certifications:

## Briefing Rules

- Keep the first pass to one product direction.
- Avoid mixing multiple unrelated products into one run.
- If pricing or MOQ is unknown, mark it unknown instead of inventing a range.
- If the operator already knows one promising keyword, start from that keyword rather than broadening immediately.

## Example

```text
Product direction: stainless steel insulated lunch box
Search keywords or synonyms: 不锈钢 保温 饭盒, 便携 保温 餐盒
Price band: 15-35 RMB ex-factory reference
MOQ preference: <= 500 if possible
Preferred supplier region: Zhejiang or Guangdong preferred
Must-have traits: can show multiple similar SKUs, visible factory or source-manufacturer signals
Must-avoid traits: only single low-detail listings, unclear supplier identity
Notes: first pass is for supplier discovery, not final procurement
```
